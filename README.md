<b>Homestead</b>  uses monetary premium and digital asset speculation to subsidize interest rates for borrowers. 

The system mints credit tokens that are backed by claims on a borrowerâ€™s real estate.  These tokens can then be sold by to purchase the home or pay off previous financing.  Over time, the borrower will repurchase the token from the market to pay off their loans.

Holders of these credit tokens are minted an ownership token.  Interest from the mortgages as well as fees inherit in the system are used to buy back and burn this ownership token.  

## Setting up and testing

It's a simple foundry repo.  

```shell
$ cd homestead
```

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```
